==============================================================
  SCRIPTURE GAME  -  Desktop Launchers
==============================================================

These helpers put a "Scripture Game" launcher on your desktop
so you can start the game with one click. They all open the
game OFFLINE in your default browser - no internet, no server.

Run the one for your system ONCE:

  Windows ...  Install Desktop Launcher - Windows.bat
  macOS .....  Install Desktop Launcher - Mac.command
  Linux .....  Install Desktop Launcher - Linux.sh

After that, just use the "Scripture Game" icon on your desktop
any time you want to play.

Prefer not to install anything? You can always launch directly
from the main folder with start-windows.bat / start-mac.command
/ start-linux.sh, or simply open index.html in any browser.
